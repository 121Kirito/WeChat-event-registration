# WeChat-event-registration
这是一个校园活动报名的微信小程序,功能未全面开发只实现了主要功能
为了应付课设，使用cursor一小时开发完成
项目架构：Spring Boot + MybatisPlus +微信小程序
项目启动需要将appID与后端appid和密钥一致
